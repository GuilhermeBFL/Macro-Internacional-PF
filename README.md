## Estrutura do guia HTML

- **7 seções temáticas** com explicação simplificada + aprofundamento
- **36 exercícios** (todas as questões das Listas 1–4 + questões das PIs 2023–2025)
- **36 gabaritos** completos com: Como ler → Passo a passo → Resposta final → Erros comuns
- **Mapas de incidência** por tópico (provas e listas)
- **SVGs inline** (mercado FX, gráfico duplo câmbio+monetário, modelo DD-AA)
- **Apêndice** com 5 questões das listas não incluídas no corpo principal
- **Anti-spoiler**: gabaritos revelados apenas ao clicar em "Ver gabarito"
- **Tooltips** para siglas (BdP, PDJ, UIP, PPC, ML...)
